# Learning Knowledge Base

Persistent, accumulating domain knowledge for the Stanford RNA 3D Folding 2 competition pipeline. Every iteration syncs new insights here so nothing is lost between sessions.

## Structure

| Directory | Scope |
|-----------|-------|
| `cs/` | Algorithms, data structures, computational geometry, complexity |
| `ml/` | Model architectures, loss functions, training techniques, evaluation metrics |
| `bio/` | RNA biology, structural bioinformatics, PDB, nucleotide chemistry |
| `sources.md` | Master bibliography of all URLs, papers, and notebooks referenced |
